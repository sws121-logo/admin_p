Get Started

How to Access the Admin Panel

Admin Access Guide

Follow these steps to access the Hotel Infinity admin panel:

1 Locate the Admin Button

Scroll to the bottom right corner of any page on the Hotel Infinity website. You will find a floating button with a shield icon.

2 Click the Admin Button

Click on the floating admin button. You will be redirected to the admin login page.

3 Enter Credentials

On the login page, enter your admin email and password. Use the demo credentials below for testing.

4 Access Admin Dashboard

After successful login, you will be redirected to the admin dashboard where you can manage the website.

Direct URL Access

You can also access the admin panel directly by appending #admin-login to the website URL:

URL Format: https://yourdomain.com#admin-login

This will take you directly to the admin login page where you can enter your credentials.

First-Time Admin Registration

If you don't have an admin account yet:

  1. Go to the admin login page
  2. Click on the "Register" link
  3. Fill in the registration form with your details
  4. Submit the form to create your admin account
  5. Use your new credentials to log in

Note: In a production environment, admin registration might be restricted to super admins.
